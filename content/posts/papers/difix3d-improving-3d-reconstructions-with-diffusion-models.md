## 1. 서론 및 배경

**요약**

Neural Radiance Fields(NeRF)와 3D Gaussian Splatting(3DGS)은 3D 재구성과 새로운 뷰 합성 분야에서 혁명을 일으켰습니다. 그러나 이들 방법들은 멀리 떨어진 뷰에서 새로운 뷰를 렌더링할 때 포토리얼한 결과를 달성하기 어려워합니다. 이 논문은 이러한 한계를 극복하기 위해 DiFix3D+라는 새로운 파이프라인을 제안합니다.

DiFix3D+의 핵심 아이디어는 두 가지입니다:
1. **DiFix**: 단일 스텝 이미지 확산 모델을 사용하여 렌더링된 뷰의 아티팩트를 제거하는 신경 향상기
2. **Progressive 3D Update**: 생성된 의사 훈련 뷰를 점진적으로 3D 표현에 다시 주입하여 품질을 개선

**핵심 개념**

- **Neural Radiance Fields (NeRF)**: 5D 좌표(공간 위치, 보기 방향)에서 색상과 밀도를 연속적으로 모델링하는 장면 표현 방식
- **3D Gaussian Splatting (3DGS)**: 3D 가우시안 입자들의 위치, 회전, 스케일 파라미터를 사용하여 장면을 표현
- **Diffusion Models**: 노이즈가 추가된 이미지에서 깨끗한 이미지를 생성하는 생성 모델
- **의사 훈련 뷰(Pseudo-training views)**: 3D 모델에서 생성된 임의의 뷰로, 학습 데이터를 증강하는 데 사용

## 2. 관련 연구

**요약**

3D 재구성의 불일치 문제를 해결하기 위한 여러 접근법들이 제안되었습니다. NeRF와 3DGS는 완벽한 입력 데이터에서는 뛰어나지만, 실세계 캡처에서는 카메라 포즈 오류, 조명 변화, 불완전한 카메라 캘리브레이션 등으로 인해 아티팩트가 발생합니다.

기존 해결책들:
- 기하학적 프라이어 도입(깊이 맵, 법선 정보 사용)
- 카메라 포즈 최적화
- 조명 변화 처리
- 생성 프라이어 활용(GAN 등)

최근에는 확산 모델의 강력한 생성 능력이 주목받고 있으며, 텍스트-이미지 생성뿐 아니라 3D 향상에도 응용되고 있습니다.

**핵심 개념**

- **Shape-Radiance Ambiguity**: 3D 장면의 기하학과 외형이 동시에 결정되는 모호성으로, 부정확한 기하학에서 그럴듯한 외형이 보간될 수 있음
- **Multi-view Consistency**: 여러 각도에서 렌더링된 뷰들이 일관성 있게 나타나야 함
- **Generative Priors**: 대규모 데이터로부터 학습된 생성 모델의 선행 정보

## 3. 배경 지식

**요약**

### 3D 장면 재구성과 새로운 뷰 합성

Neural Radiance Field(NeRF)는 장면의 색상을 다음과 같이 모델링합니다:

$$C(p) = \sum_{i=1}^{N} c_i \alpha_i \prod_{j=1}^{i-1} (1 - \alpha_j)$$

**수식 설명**
- **$C(p)$**: 픽셀 $p$에 대한 최종 출력 색상
- **$c_i$**: 광선을 따라 $i$번째 샘플 위치의 색상
- **$\alpha_i$**: $i$번째 샘플의 불투명도 (0 = 완전 투명, 1 = 완전 불투명)
- **$\sum$**: 광선을 따라 모든 샘플들의 색상을 누적
- **$\prod_{j=1}^{i-1}(1 - \alpha_j)$**: $i$번째 샘플까지 도달한 빛의 투과율
  - 예: 앞의 두 샘플이 각각 0.5 불투명이면, 세 번째 샘플은 0.25 강도로만 보임

### 3D Gaussian Splatting

3DGS는 각 가우시안을 다음과 같이 정의합니다:

$$\alpha_i = \eta \cdot \exp\left(-\frac{1}{2}(p - \mu_i)^T \Sigma_i^{-1}(p - \mu_i)\right)$$

**수식 설명**
- **$\alpha_i$**: $i$번째 가우시안의 불투명도
- **$\eta$**: 학습된 기본 불투명도 값
- **$\mu_i$**: 가우시안의 중심 위치
- **$\Sigma_i$**: 가우시안의 공분산 행렬 (형태를 정의)
- **$(p - \mu_i)^T \Sigma_i^{-1}(p - \mu_i)$**: 마할라노비스 거리로, 점 $p$가 가우시안 중심에서 얼마나 멀리 있는지 측정
  - 거리가 크면 불투명도는 급격히 감소 (종 모양 곡선)

### 확산 모델

확산 모델은 노이즈가 있는 이미지를 점진적으로 정제합니다:

$$\mathcal{L}_{\text{score}} = \mathbb{E}_{x, c, \tau, \epsilon \sim \mathcal{N}(0,I)} \left[ \left\| y - F_\theta(x_\tau; c, \tau) \right\|^2 \right]$$

**수식 설명**
- **$x_\tau$**: $\tau$ 스텝에서의 노이즈가 있는 이미지 ($x_\tau = \alpha_\tau x + \sigma_\tau \epsilon$)
- **$F_\theta$**: 학습된 노이즈 예측 네트워크
- **$c$**: 선택적 조건 정보 (예: 참조 뷰)
- **$\epsilon$**: 표준 정규분포에서 샘플링된 노이즈
- **이 손실함수의 의미**: 모델이 각 노이즈 레벨에서 정확하게 노이즈를 예측하도록 학습

**핵심 개념**

- **Denoising Score Matching**: 노이즈 제거 과정에서 모델을 학습하는 방법
- **Conditioning**: 확산 모델에 추가 정보(참조 이미지 등)를 제공하여 더 나은 제어
- **Uniform Distribution over Diffusion Time**: 모든 노이즈 레벨에서 균등하게 학습하여 일반화 성능 향상

## 4. DiFix를 통한 3D 재구성 개선

**요약**

### 4.1 DiFix: 사전학습된 확산 모델에서 3D 향상기로

DiFix의 기본 아이디어는 단일 스텝 확산 모델(SD-Turbo)을 이용하여 NeRF/3DGS 렌더링의 아티팩트를 제거하는 것입니다.

**입력**: 노이즈가 있는 렌더링 이미지, 참조 뷰들
**처리 과정**:
1. SD-Turbo를 기본으로 사용하되, 낮은 노이즈 레벨($\tau = 200$)로 미세 조정
2. 참조 뷰를 조건으로 사용하여 일관성 유지
3. 참조 뷰와 렌더링된 뷰 간의 교차 의존성을 학습

**손실 함수**

재구성 손실:
$$\mathcal{L}_{\text{recon}} = \left\| \tilde{I} - I \right\|_2$$

지각적 손실(LPIPS):
$$\mathcal{L}_{\text{LPIPS}} = \frac{1}{L} \sum_{l=1}^{L} \left\| \phi_l(I) - \phi_l(\hat{I}) \right\|_2$$

**수식 설명**
- **$\tilde{I}$**: 모델의 출력 이미지
- **$I$**: 정답 이미지
- **$\phi_l$**: 사전학습된 VGG-16 네트워크의 $l$번째 레이어의 특징 맵
- LPIPS는 인간의 지각과 더 잘 맞는 지각적 유사성을 측정
  - 픽셀 수준의 차이보다 의미 있는 차이를 더 잘 포착

스타일 손실(Gram Matrix 기반):
$$\mathcal{L}_{\text{Gram}} = \frac{1}{L} \sum_{l=1}^{L} \left\| G_l(I) - G_l(\hat{I}) \right\|_2$$

**수식 설명**
- **$G_l(I)$**: $l$번째 레이어 특징의 그람 행렬
- 그람 행렬은 특징 간의 상관관계를 나타내며, 이미지의 질감과 스타일을 포착
- 따라서 이 손실은 더 선명하고 세부적인 이미지를 생성하도록 격려

최종 손실:
$$\mathcal{L} = \mathcal{L}_{\text{recon}} + \mathcal{L}_{\text{LPIPS}} + 0.5\mathcal{L}_{\text{Gram}}$$

### 4.2 DiFix3D+: 확산 프라이어를 사용한 새로운 뷰 합성

DiFix3D+의 핵심은 **점진적 3D 업데이트** 파이프라인입니다:

**단계 1: 초기 NeRF 최적화**
- 참조 뷰들로부터 3D 표현 학습

**단계 2: 의사 훈련 뷰 생성**
- 참조 뷰 근처의 중간 뷰들을 렌더링
- DiFix를 사용하여 아티팩트 제거
- 정제된 뷰들을 훈련 세트에 추가

**단계 3: 점진적 카메라 포즈 섭동**
- 각 반복마다 1.5k 인터레이션마다 카메라를 조금씩 이동
- 렌더링된 뷰를 DiFix로 정제하고 훈련 세트에 추가
- 점진적으로 더 먼 뷰로 확장

**수식 설명**

점진적으로 훈련 세트를 확대하면:
1. 모델이 점차 더 많은 뷰에 대해 일관성 있는 3D를 학습
2. 멀리 떨어진 뷰에서도 정확한 재구성 가능
3. 제한된 캡처 범위에서도 확장된 범위의 뷰 합성 가능

**핵심 개념**

- **Reference View Conditioning**: 참조 뷰를 확산 모델의 조건으로 사용하여 외형 일관성 유지
- **Reference Mixing Layer**: 참조 뷰의 특징을 중간에 주입하여 정보 효율성 증대
- **Data Curation**: 주요 아티팩트(희소 재구성, 사이클 재구성, 교차 참조, 모델 언더피팅)를 시뮬레이션하여 다양한 훈련 데이터 생성

## 5. 실험 결과

**요약**

### 5.1 야생 장면에서의 아티팩트 제거 (DL3DV 데이터셋)

**정량적 결과 (Table 2)**

| 메서드 | PSNR↑ | SSIM↑ | LPIPS↓ | FID↓ |
|------|-------|-------|--------|------|
| Nerfacto | 19.08 | 0.6633 | 0.3277 | 63.77 |
| DiFix3D (NeRF) | 18.32 | 0.6623 | 0.2789 | 49.44 |
| **DiFix3D+ (3DGS)** | **18.51** | **0.6858** | **0.2637** | **41.77** |
| 3DGS | 17.66 | 0.6780 | 0.3265 | 113.84 |

**결과 해석**:
- DiFix3D+는 LPIPS에서 19.4% 개선 (0.3277 → 0.2637)
- FID 스코어에서 34.5% 개선으로 시각적 품질이 크게 향상
- PSNR은 약간 감소했지만, 이는 더 사실적인 세부 표현으로 인한 것

### 5.2 자동주행 장면 향상 (RDS 데이터셋)

**정량적 결과 (Table 3)**

| 메서드 | PSNR↑ | SSIM↑ | LPIPS↓ | FID↓ |
|------|-------|-------|--------|------|
| Nerfacto | 19.95 | 0.4930 | 0.5300 | 91.38 |
| Nerfacto + DiFix3D | 21.52 | 0.5700 | 0.4266 | 77.83 |
| Nerfacto + DiFix3D+ | **21.75** | **0.5829** | **0.4016** | **73.08** |

**결과 해석**:
- SSIM 개선: 0.4930 → 0.5829 (18.2% 향상)
- LPIPS 개선: 0.5300 → 0.4016 (24.2% 향상)
- FID 개선: 91.38 → 73.08 (19.9% 향상)

### 5.3 성분별 절제 연구 (Table 5 - Nerfacto 데이터셋)

| 성분 | PSNR↑ | SSIM↑ | LPIPS↓ | FID↓ |
|-----|-------|-------|--------|------|
| 기준선 (Nerfacto) | 19.29 | 0.6214 | 0.3903 | 134.65 |
| + 노이즈 감소 | 19.36 | 0.6217 | 0.3865 | 134.02 |
| + 참조 뷰 조건 | 19.50 | 0.6289 | 0.3682 | 115.08 |
| + Gram 손실 | 19.74 | 0.6563 | 0.3424 | 75.94 |
| **+ 3D 업데이트** | **18.32** | **0.6623** | **0.2789** | **49.44** |

**결과 해석**:
- 각 성분이 점진적으로 성능 향상에 기여
- 3D 점진적 업데이트가 가장 큰 개선 (FID: 75.94 → 49.44)
- 노이즈 감소 혼자로는 미미한 효과, 조건화와 손실이 핵심

### 5.4 실시간 성능

- **추론 시간**: NVIDIA A100 GPU에서 이미지당 76ms (단계별 확산 모델의 10배 빠름)
- **메모리 효율성**: 실시간 처리에 적합한 경량 모델

**핵심 개념**

- **PSNR (Peak Signal-to-Noise Ratio)**: 이미지 복원 품질의 전통적 지표 (높을수록 좋음)
  - 공식: $PSNR = 10 \log_{10}\left(\frac{MAX^2}{MSE}\right)$ (MAX=255 for 8-bit images)
  - 한계: 픽셀 수준 차이에만 민감하여 지각적 차이를 반영하지 못함

- **SSIM (Structural Similarity Index)**: 구조적 유사성 측정 (0~1, 높을수록 좋음)
  - 밝기, 대비, 구조를 모두 고려
  - PSNR보다 인간 지각과 더 잘 맞음

- **LPIPS (Learned Perceptual Image Patch Similarity)**: 지각적 손실 (낮을수록 좋음)
  - 신경망의 특징을 사용하여 의미 있는 차이 측정
  - 인간의 시각적 판단과 가장 잘 일치

- **FID (Fréchet Inception Distance)**: 생성 이미지 품질 평가 (낮을수록 좋음)
  - 실제 이미지와 생성 이미지의 특징 분포 간 거리
  - 시각적 충실도와 다양성을 평가

## 6. 주요 기여 및 혁신

**요약**

DiFix3D+는 다음과 같은 주요 혁신을 제시합니다:

1. **단일 스텝 확산 모델을 신경 향상기로 활용**
   - 기존 멀티스텝 확산 모델 대비 10배 빠른 추론
   - 미세 조정으로 렌더링 아티팩트 효과적 제거

2. **점진적 3D 업데이트 파이프라인**
   - 의사 훈련 뷰를 점진적으로 3D 표현에 재주입
   - 멀리 떨어진 뷰에서도 일관성 있는 결과 달성

3. **자체 조건화 메커니즘**
   - 참조 뷰를 조건으로 사용하여 외형 일관성 유지
   - Reference Mixing Layer로 효율성 극대화

4. **포괄적 데이터 큐레이션**
   - 실제 아티팩트를 시뮬레이션하여 다양한 훈련 데이터 생성
   - NeRF와 3DGS 모두에 호환

5. **정량적 성능 향상**
   - FID 스코어에서 평균 2배 향상
   - LPIPS와 시각적 품질에서 유의미한 개선
   - 3D 일관성 유지

## 핵심 개념 정리

### 렌더링 아티팩트와 불일치

| 아티팩트 유형 | 원인 | DiFix3D+ 해결법 |
|-------------|-----|----------------|
| 블러 | 부정확한 기하학 | 의사 뷰로 기하학 개선 |
| 유령 효과 | 멀티뷰 불일치 | 조건화로 참조 뷰 기준 설정 |
| 누락된 영역 | 불완전한 캡처 | 점진적 뷰 확대로 외삽 |
| 색상 변이 | 조명 변화 | Gram 손실으로 색상 안정화 |

### 확산 모델의 역할

확산 모델이 효과적인 이유:
1. **사전학습된 생성 프라이어**: 대규모 자연 이미지에서 학습한 일반적 지식
2. **적응적 노이즈 처리**: 다양한 노이즈 레벨에서 효과적으로 작동
3. **조건화 가능성**: 참조 정보를 통한 유연한 제어
4. **고속 추론 가능**: 단일 스텝 모델 사용으로 실시간 처리

### 점진적 업데이트의 장점

```
반복 1: 참조 뷰 근처의 의사 뷰 생성
  ↓
반복 2: 더 먼 각도의 의사 뷰 생성
  ↓
반복 3: 극단적 각도까지 점진적 확장
  ↓
결과: 전체 뷰포트에 일관된 3D
```

## 결론 및 시사점

### 논문의 기여

DiFix3D+는 3D 재구성과 새로운 뷰 합성의 품질 향상을 위한 실용적이고 효과적인 솔루션을 제시합니다. 특히:

1. **실시간 처리 가능**
   - 단일 스텝 확산 모델로 76ms 추론 시간 달성
   - 기존 멀티스텝 확산 모델의 10배 빠름

2. **호환성과 일반성**
   - NeRF와 3DGS 모두에 적용 가능
   - 다양한 장면 타입(실내, 실외, 자동주행)에서 작동

3. **정량적 개선**
   - FID 스코어 2배 향상
   - LPIPS 20-25% 향상
   - 시각적 아티팩트 효과적 제거

### 한계 및 향후 연구 방향

**현재 한계**:
- 3D 재구성이 완전히 실패한 경우는 여전히 개선 어려움
- 초기 기하학의 품질에 의존
- 극단적인 뷰 각도에서의 외삽에는 여전히 제약

**향후 가능성**:
1. 현대 확산 모델 프라이어 통합
   - 더 최신의 생성 모델 사용으로 품질 향상

2. 비디오 확산 모델로 확장
   - 시간적 일관성을 고려한 동영상 3D 재구성

3. 더 효율적인 실시간 처리
   - 더 빠른 하드웨어나 경량화된 모델로 추론 시간 단축

### 실무적 시사점

1. **3D 콘텐츠 생성 파이프라인**
   - 영화, 게임, VR 콘텐츠 제작에 활용 가능
   - 촬영 범위의 제약을 극복하여 비용 절감

2. **로봇 비전 시스템**
   - 자동주행 차량의 3D 환경 이해 향상
   - 실시간 처리로 현장 적용 가능

3. **의료 영상 처리**
   - CT, MRI 스캔의 품질 향상
   - 부분적 스캔으로도 완전한 3D 재구성 가능

4. **문화유산 디지털화**
   - 박물관 유물의 고품질 3D 스캔
   - 접근성 제약을 극복한 가상 전시 가능
